modules.run.shell

Run a shell command

Examples

_: Execute raw shell command
run.shell:
  command: echo 'hello itsudeno!'

_: Execute raw shell command with environment variables
run.shell:
  command: echo 'hello $ITSUDENO!'
  environment:
    ITSUDENO: itsudeno

Arguments

command Alias: cmd Required

string
Command to run

environment Alias: env Default: {}

string{}

cwd

This module does not collect any past state
