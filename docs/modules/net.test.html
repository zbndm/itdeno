modules.net.test

Test connection with a remote host (and eventually a remote port)

Examples

_: Test domain port
net.test:
  host: itsudeno.land
  port: 443

_: Test target host ssh connection
using: local
net.test:
  host: '${it.target.executors.ssh?.host}'
  port: '${it.target.executors.ssh?.port}'

Arguments

host Default: "localhost"

string
Remote host

port

number port
Remote port

Past state

This module does not collect any past state

Result

This module does not return any result

Supported platforms

This module contains a generic implementation which should work on most operating systems.

Specific implementations:

  • windows
