modules.log

Log a message

This module is always executed on controller, regardless of current executor.

Examples

_: Print current time
targets: (localhost)
log:
  message: 'Current time is ${new Date()}'

_: Print current host data
targets: (localhost)
log:
  message: '${it.inspect(it.target)}'

Arguments

message Alias: msg Required

string
Message to log

Past state

This module does not collect any past state

Result

message

string
Logged message

Supported platforms

This module contains a generic implementation which should work on most operating systems.
