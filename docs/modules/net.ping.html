modules.net.ping

Ping an host using ICMP

Examples

_: Ping target host
using: local
net.ping:
  host: '${it.target.name}'

_: Ping ip address
net.ping:
  host: 127.0.0.1

_: Ping domain
net.ping:
  host: itsudeno.land

Arguments

host Required

string
Host to ping

packets Default: 4

number positive integer nonzero
Number of packets to send

Past state

This module does not collect any past state

Result

host

string
Pinged host

ip Optional

string
Pinged host ip address (may be null if dns resolution failed)

transmitted

number positive integer
Number of transmitted packets

received

number positive integer
Number of received packets

loss

number percentage
Packets loss percentage

Supported platforms

This module contains a generic implementation which should work on most operating systems.

Specific implementations:

  • windows
